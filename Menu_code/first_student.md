# CÁC LỆNH SQL quan trọng nhất 
`SELECT` - trích xuất dữ liệu từ ` cơ sở dữ liệu `
`UPDATE` - cập nhập dữ liệu trong ` CSDL `
`DELETE` - Xoá dữ liệu khỏi 
`INSERT INTO` - chèn dữ liệu mới vào
===================================================
`CREATE DATABASE` - tạo `CSDL` mới
`ALTER DATABASE` - Chỉnh sửa

===================================================

` CREATE TABLE ` - tạo 1 bảng mới 
`ALTER DATABASE `- chỉnh sửa bảng
`DROP TABLE`     - xoá 1 bảng 
===================================================

`CREATE INDEX` - tạo chỉ mục (khoá tìm kiến)
`DROP INDEX ` - xoá 1 chỉ mục
